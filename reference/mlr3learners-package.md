# mlr3learners: Recommended Learners for 'mlr3'

More learners are implemented in the [mlr3extralearners
package](https://github.com/mlr-org/mlr3extralearners). A guide on how
to create custom learners is covered in the book:
<https://mlr3book.mlr-org.com>. Feel invited to contribute a missing
learner to the [mlr3](https://CRAN.R-project.org/package=mlr3)
ecosystem!
